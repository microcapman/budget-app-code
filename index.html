Mark Braun Budget App

This application is a private personal budgeting tool used by Mark Braun and household members.

Data Usage

This app connects to financial institutions using Plaid to retrieve account and transaction data. The data is used solely for personal budgeting, spending analysis, and financial planning.

Data Storage

Data is stored securely in a private PostgreSQL database and is not sold or shared with third parties.
